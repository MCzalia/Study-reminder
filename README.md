PROJECT TITLE: Student Reminder Website

	Project Idea
Project Name: Smart Study Reminder
Problem:
Many students forget assignment deadlines and exam dates.
Solution:
Create a simple website where students can:
•	Add assignments
•	Set deadlines
•	View reminders
•	Track completed tasks
	Research (5 Students)
I asked 5 students the following questions:
Questions Asked:
1.	Do you forget assignment deadlines?
2.	How do you currently track assignments?
3.	Would you use a website to manage tasks?
4.	What features would you like?
5.	Do you prefer mobile-friendly design?

Research Results:
Question	Result
Forget deadlines?	4 out of 5 said YES
Current tracking	WhatsApp group & notebook
Use website?	All 5 said YES
Wanted features	Reminder alerts, simple interface
Mobile design?	All said VERY IMPORTANT

Conclusion from Research:
Students need:
•	Simple design
•	Mobile responsive layout
•	Deadline reminders
•	Easy task adding system
